## Internal Roles (Tenant Users)

### TENANT ADMIN (Analytics Context)
- Full access to all reports
- Configure dashboards
- Export data
- Create custom reports

### TENANT USER (Analytics Context)
- View assigned reports/dashboards
- ❌ Cannot configure reports
- ❌ Cannot export sensitive data
